DEMO


$('.scramble-default').scramble({framePerReveal: 4});

This will be unscrambled using the default seed.

Beware non-monospaced fonts, because this is what it looks like on "Open Sans".


$('.scramble-une-pipe').scramble({framePerReveal: 4, seed: '____|'});

Unscrambling this will be done with only a bunch of underscores and a pipe.


$('.scramble-kana').scramble({
	framePerReveal: 4,
	seed: 'あいうえおかきくけこさしすせそたちつてとなにぬねのはひふへほまみむめもやゆよらりるれろわを'});
});

And lastly special consideration needs to be given to using non-latin characters when unscrambling, such as hiragana ones.
